# M122

# 04-FileDiff
Sie können das Projekt mit 04-FileDiff.sh file1 file2 laufen lassen. 
Sie müssen die drei Files lokal bei sich haben, ansonsten geht das Skript nicht.

# 05-Zahlenraten
Für das Zahlenraten brauchen Sie nur zahlenraten.sh Die Erklärung und Anleitung zum
Spiel finden Sie im Skript

# 06-Umgestalten
Für das Umgestalten brauchen Sie das File umgestalten.sh ExportSmall.spl und output.txt
Das Skript können Sie mit `bash umgestalten.sh ExportSmall.spl output.txt` starten. 
Den Output des Skriptes finden Sie dann im output.txt
